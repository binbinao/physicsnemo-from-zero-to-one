# 代理模型选型决策树（CAE）

```text
需要回答的设计问题？
├─ 工况数 < 10 且几何固定 → 直接跑传统 CFD/FEM
├─ 需要反演材料/边界（少数据）→ PINN（ch01–03）
├─ 参数化几何/工况 > 100，有 CFD 样本 → FNO/MLP 代理（ch04–07）
└─ 签审/sign-off → 必须高保真复核，代理仅作筛选
```

| 方法 | 数据 | 训练成本 | 推理 | 外推风险 |
|:---|:---|:---|:---|:---|
| 全量 CFD | 0 ML | 高/工况 | 高 | 低（若收敛） |
| PINN | 0–少 | 中–高 | 低 | 中（几何复杂时） |
| 纯数据 FNO | 多 | 高（一次性） | 极低 | 高 |
| 数据+物理 FNO | 中 | 高 | 极低 | 中 |

详见各章行业映射与 [CAE_CONSERVATION_CHECKLIST.md](CAE_CONSERVATION_CHECKLIST.md)。
