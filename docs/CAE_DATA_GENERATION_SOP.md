# CAE 训练数据生成 SOP（骨架）

> 本书微缩实验用合成数据；**落地项目**按此清单对接真实求解器。

## 1. 需求与 DOE

- [ ] 设计变量范围与制造/法规约束  
- [ ] DOE 方法（LHS / 全因子 / 自适应）  
- [ ] 目标场与标量（$C_d$, $T_{max}$, 压力场等）

## 2. 几何与网格

- [ ] CAD 版本（PLM 编号）  
- [ ] 网格类型（多面体/六面体/边界层）  
- [ ] **网格收敛**：至少 2 套 refinement，记录 $y^+$/单元数  

## 3. 求解器设置

- [ ] 软件与版本（Fluent 2024 R1 / STAR-CCM+ 18.06 / Icepak …）  
- [ ] 物理模型（稳态/瞬态、湍流模型、k-ω SST …）  
- [ ] 边界条件与材料 **SI 单位**  
- [ ] 收敛准则与残差截图归档  

## 4. 输出契约

```text
dataset/
  meta.json          # solver, turbulence, mesh level, git commit
  case_0001/
    params.json
    fields/          # npz/vtk, 与训练脚本约定一致
```

## 5. 质量门

- [ ] 失败工况剔除规则  
- [ ] 与实验/风洞 spot check（若有）  
- [ ] 训练/验证/测试 **几何 holdout**（防泄漏）

## 6. 网格收敛对代理模型的影响（C16）

代理模型学到的是 **「该网格密度下求解器的映射」**。混用不同 refinement 会导致非物理跳变。建议：**单一网格等级** 生成主数据集，或把 `mesh_level` 作为显式输入。

## 7. 多保真（C17）

低保真（面元法/粗网格）+ 高保真（RANS）可分层训练；**本书未提供多保真代码**，仅 ch04 概念叙述。
